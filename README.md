‚úÖ Project Overview

The Sales Performance Dashboard created in Tableau is designed to analyze and monitor the sales performance of a business across different time periods, regions, categories, and customer segments.
This dashboard helps management understand which products are selling well, which markets are performing strongly, and how revenue trends are changing over time.

The visualization enables decision-makers to:

Track KPI performance

Identify growth opportunities

Manage inventory more efficiently

Set strategic business targets
